Sampler Web Component - Demonstration

Point 11 : Web Component

Cette page demonstre l'utilisation du sampler comme un composant web reutilisable.

Code HTML simple : <sampler-component></sampler-component>

Avantages du Web Component

  • Encapsulation : Le CSS et le DOM sont isoles du reste de la page
  • Reutilisabilite : Utilisable sur plusieurs pages ou projets
  • API propre : Methodes publiques (loadPreset, playPad, stopAll)
  • Shadow DOM : Pas de conflits de styles ou de scripts
  • Standard Web : Fonctionne sans framework (Vanilla JS)
